# Flow Cytometer Booking System: HANS 305

## Please read
Anyone can modify a booking slot until the time has passed. This is an honor-based system, please ***do not cancel*** other people's bookings without checking with them first.
If misuse becomes an issue, access restrictions may need to be introduced 🥺

## Features
- 📅 Weekly calendar view with 30-minute time slots
- 📆 Book up to 1 month in advance
- 🔒 Required fields: Name and PI/Lab name
- 🛡️ Admin panel (password-protected) to manage bookings and block out maintenance windows
