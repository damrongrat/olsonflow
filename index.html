How sync works
  • When a booking is made → event is created on the calendar
  • When a booking is cancelled → event is deleted from the calendar
  • Requires an API key with Google Calendar API enabled
  • The calendar must be shared with the service account or set to public
  • If no API key is set, a manual "Add to Google Calendar" link is shown instead
